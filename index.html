Projects

HireHelper

Created during SheInnovates 2026 Hackathon, the HireHelper web app prepares people for interviews and job applications.

The user can input their resume, linkedin, and specify which job or field they are applying for.

The site will generate ratings for the resume and linkedin, and provide mock interview questions with feedback.

The frontend was created using HTML and Javascript, while the backend was created using Python.

uMIPS CPU

Simulated Single-Cycle CPU with 8 16-bit register, and various 16 bit instructions

Contains an program counter, instruction memory, register file, control, ALU, and a second memory for writing

Created using Logisim circuit software, and loosely based off of MIPS assembly programming language
